Практическеи работы по Java с использованием Docker-контейнера

Использование возможно без применения Docker. Требуется наличие JDK 21. Возможность запуска исполняемого файла проверялась на системе Windows 10 via openJDK, Ubuntu 20.04 via WSL, Терминальный запуск внутри контейнера via VSCode

В практических работах мною используется готовый Docker-образ eclipse-temurin:21-jdk-alpine с подключенными плагинами для VSCode

## Структура
- `practice one/` - Корень практической работы
   - 'MatrixApp.java' - Исполняемая программа. 
   - 'Readme.md' - файл с обобщающей информацией
